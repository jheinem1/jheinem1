Metatable

A metatable is a table full of metamethods which overload the functionality of existing types in Luau. Under the hood, every operator in Luau is essentially a function call. Metatables are primarily used to add functionality to Userdata, which can easily be implemented in C++.

Metamethods

Metamethods are functions within metatables such as __index and __add that define the behavior of operators in Luau acting on a given type.

Interaction

lang.luau.globals.setmetatable (Private) and getmetatable are primarily used to interact with metatables.
