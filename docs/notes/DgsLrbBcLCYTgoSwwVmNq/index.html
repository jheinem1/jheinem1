Luau

Luau (lowercase u, /ˈlu.aʊ/) is a fast, small, safe, gradually typed embeddable scripting language derived from Lua.

Luau is essentially a superset of lang.lua (Private) 5.1, including similar type syntax to lang.typescript (Private).
