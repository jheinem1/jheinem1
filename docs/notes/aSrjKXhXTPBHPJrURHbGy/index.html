newproxy

newproxy(addMetatable=false): userdata

Creates and returns a new userdata with or without a metatable. If created with a metatable it can be readily modified with getmetatable, and if not, will have to be created using setmetatable.

Examples

local myUserdata = newproxy(true)
getmetatable(myUserdata).__index = {1, 2, 3}
print(type(myUserdata)) --> "userdata"
print(myUserdata[1]) --> 1
